## 🎯 核心功能说明

### 工作流程

1. **启动阶段**
   - 加载配置文件
   - 初始化 AI 客户端 (AIClient)
   - 初始化知识库(向量化文本)
   - 创建/加载 AI 助手
   - 启动 Flask 服务

2. **对话阶段**
   - 接收用户消息
   - 创建/获取会话线程
   - 调用 AI 助手处理
   - 检测工具调用需求
   - 执行工具(如表单提交)
   - 返回 AI 回复

3. **工具调用**
   - AI 判断是否需要调用工具
   - 解析工具参数
   - 执行对应功能
   - 将结果返回 AI
   - 生成最终回复

### API 接口

#### GET /start
检查服务状态

**请求:**
```
GET http://localhost:5000/start
```

**响应:**
```json
{
  "status": "ready",
  "message": "客服系统已就绪",
  "assistant_id": "asst_xxx",
  "knowledge_base": true
}
```

#### POST /chat
处理对话请求

**请求:**
```json
{
  "message": "用户输入",
  "session_id": "可选会话ID"
}
```

**响应:**
```json
{
  "reply": "AI回复内容",
  "tool_calls": [],
  "session_id": "会话ID",
  "finished": false
}
```

## 🔧 技术栈

- **后端框架**: Flask 3.0.0
- **AI SDK**: OpenAI 1.12.0 (兼容多种 AI 服务)
- **向量化**: text-embedding-v3
- **AI 模型**: deepseek-v4-pro
- **HTTP 客户端**: requests 2.31.0
- **数值计算**: numpy 2.4.4
- **环境管理**: python-dotenv 1.0.0

## 🚀 使用方法

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置 API Key
编辑 `config.json` 文件,填入 API Key

### 3. 准备知识库
编辑 `knowledge_base/raw_knowledge.txt`,添加业务知识

### 4. 启动服务
```bash
python app.py
```

### 5. 测试
```bash
# 另一终端运行
python test.py
```

## 🎨 自定义扩展

### 修改助手指令
编辑 `modules/prompt/__init__.py` 中的 `CUSTOMER_SERVICE_PROMPT` 常量

### 添加新工具
1. 在 `modules/tools/` 目录添加工具文件
2. 在 `__init__.py` 中注册工具
3. 更新工具定义

### 更新知识库
1. 修改 `knowledge_base/raw_knowledge.txt`
2. 删除 `db/vector_data.json`
3. 重启服务
