# 智能客服 Agent - 快速启动指南

## 📋 前置准备

### 1. 获取阿里云百炼 API Key

1. 访问: https://bailian.console.aliyun.com/
2. 登录阿里云账号
3. 进入 API-KEY 管理
4. 创建新的 API Key
5. 复制保存

### 2. 配置 API Key

打开 `.env` 文件,填入您的 API Key:

```env
DASHSCOPE_API_KEY=您的API_Key_粘贴到这里
```

## 🚀 启动服务

### 方式一:直接启动

```bash
python app.py
```

### 方式二:使用测试脚本

先在一个终端启动服务:
```bash
python app.py
```

然后在另一个终端运行测试:
```bash
python test.py
```

## 📡 API 测试

### 1. 检查服务状态

浏览器访问: http://localhost:5000/start

或使用 curl:
```bash
curl http://localhost:5000/start
```

### 2. 发送对话请求

使用 curl:
```bash
curl -X POST http://localhost:5000/chat ^
  -H "Content-Type: application/json" ^
  -d "{\"message\": \"你好,我想了解一下你们的产品\"}"
```

使用 Python:
```python
import requests

response = requests.post(
    'http://localhost:5000/chat',
    json={
        'message': '你好,我想了解一下你们的产品'
    }
)
print(response.json())
```

## 📝 自定义配置

### 修改知识库

编辑 `data/raw_knowledge.txt` 文件,添加您的业务知识。

格式:
- 使用空行分隔不同主题
- 可以包含:公司介绍、产品信息、常见问题等

### 修改助手指令

编辑 `assistant.json` 文件中的 `instructions` 字段。

### 配置 Airtable 表单(可选)

在 `.env` 文件中添加:
```env
AIRTABLE_API_TOKEN=your_token
AIRTABLE_BASE_ID=your_base_id
AIRTABLE_TABLE_NAME=YourTable
```

## 🔧 常见问题

### Q: 启动时提示 API Key 错误?
A: 检查 `.env` 文件是否正确配置了 `DASHSCOPE_API_KEY`

### Q: 向量化失败?
A: 检查网络连接,确认 API Key 有 embedding 权限

### Q: 如何重置知识库?
A: 删除 `knowledge_base.json` 文件,重启服务即可
