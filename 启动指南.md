# 智能客服系统 - 前后端分离版本

## 项目结构

```
customer/
├── backend/              # Python 后端 (Flask)
│   ├── app.py           # Flask 主应用
│   ├── config.json      # 配置文件
│   ├── requirements.txt # Python 依赖
│   ├── .env             # 环境变量
│   ├── modules/         # 后端模块
│   └── ...
│
└── frontend/            # React 前端 (Vite)
    ├── src/
    │   ├── components/  # UI 组件
    │   ├── api/         # API 调用
    │   └── ...
    └── package.json
```

## 启动指南

### 1. 启动后端服务

```bash
cd backend
pip install -r requirements.txt
python app.py
```

后端将运行在: `http://localhost:5000`

### 2. 启动前端服务

打开新终端：

```bash
cd frontend
npm install
npm run dev
```

前端将运行在: `http://localhost:5173`

### 3. 访问应用

在浏览器中打开: `http://localhost:5173`

## 功能说明

- **顶部**: 显示"智能客服"标题和图标
- **中间**: 聊天内容区域
  - 左侧: 客服回复（白色气泡）
  - 右侧: 用户消息（紫色渐变气泡）
- **底部**: 输入框和发送按钮
  - 支持回车发送
  - 支持点击按钮发送

## 初始状态

进入应用后，会自动显示欢迎消息：
> "您好！我是智能客服，有什么可以帮助您的吗？"

## API 接口

- `GET /start` - 检查服务状态
- `POST /chat` - 发送对话请求

## 技术栈

**后端**:
- Flask - Web 框架
- Flask-CORS - 跨域支持
- OpenAI API - AI 对话

**前端**:
- React 18 - UI 框架
- Vite - 构建工具
- Axios - HTTP 请求
- UUID - 会话管理
